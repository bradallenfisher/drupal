INTRODUCTION
------------

This module creates a better, fancier interface for the
drupal local tasks block.

REQUIREMENTS
------------

This module does not have any dependency on any other module.

INSTALLATION
------------

By downloading and enabling this module styling will be applied.
Ensure you have permission to 'contextual links'.

CONFIGURATION
-------------

This module does not have any configuration.
